To download and run the To-Do list website, refer to the following instructions.
The instructions use pythons built in http.server module.

1. Download the files and save them in the same directory. Make sure the only contents within the directory is the downloaded files.
2. Open command prompt
3. Navigate to the directory containing the project files.
4. Run the command "python -m http.server"
5. Go to your desired web browser, and go to "http://localhost:8000"
6. When finished and ready to stop the server, go back to command prompt and press "Ctrl + C"
